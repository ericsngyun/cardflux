# TCG Card Specifications - Comprehensive Reference

## Purpose
Reference document for card number positioning, formats, and distinguishing features across all major Trading Card Games. Used to build a universal card identification system.

---

## 1. Magic: The Gathering (MTG)

### Card Number Location
- **Position:** Bottom-center of card
- **Format:** `###/###` or `###` (e.g., `247/302`, `42`)
- **Additional Info:** Set code in bottom-left (3-letter, e.g., `MOM`, `BRO`)
- **Collector Info:** Artist name, copyright, set symbol

### Card Number Formats
```
Standard:     247/302
Showcase:     247s/302
Extended Art: 247a/302
Promo:        247p
```

### Set Code Position
- **Location:** Bottom-left corner
- **Format:** 3-4 uppercase letters (e.g., `MOM`, `LCI`, `WOE`)

### Variants & Special Versions
- **Foil:** Holographic sheen, star icon next to set symbol
- **Showcase:** Alternate art frame style
- **Extended Art:** Art extends to borders
- **Borderless:** No border, full art
- **Etched Foil:** Matte foil finish (CMR onwards)
- **Phyrexian:** Phyrexian language text
- **Serialized:** Numbered 001/500 style (very rare)

### Visual Distinguishing Features
- **Color Identity:** Border color (White, Blue, Black, Red, Green, Colorless, Gold)
- **Card Frame:** Modern (8th edition+), old frame, showcase frames
- **Mana Symbols:** Upper-right corner
- **Type Line:** Middle of card (Creature, Instant, Sorcery, etc.)

### Edge Cases
- **Double-faced cards:** Two card numbers (front/back)
- **Split cards:** Single number, two halves
- **Promo cards:** May have special star or "P" marker
- **Secret rares:** Number > set size (e.g., 303/302)

## 2. Yu-Gi-Oh!

### Card Number Location
- **Position:** Bottom-right corner (very small text)
- **Format:** `XXX-XX###` (e.g., `POTE-EN001`, `LOB-001`)
- **Additional Info:** Set abbreviation + language + number

### Card Number Formats
```
English:     POTE-EN001
German:      POTE-DE001
French:      POTE-FR001
1st Edition: Has "1st Edition" stamp
Unlimited:   No stamp
```

### Set Code Position
- **Location:** Bottom-right, embedded in card number
- **Format:** 3-4 letter set code (e.g., `LOB`, `POTE`, `MAMA`)

### Variants & Special Versions
- **Rarity Markers:**
  - **Common:** No foil
  - **Rare:** Silver name
  - **Super Rare:** Foil art
  - **Ultra Rare:** Foil art + gold name
  - **Secret Rare:** Foil everything + rainbow holographic
  - **Ghost Rare:** 3D holographic (discontinued)
  - **Starlight Rare:** Starburst pattern
  - **Ultimate Rare:** Embossed art texture
  - **Collector's Rare:** Purple foil
  - **Quarter Century Secret:** 25th anniversary foil

### Visual Distinguishing Features
- **Card Border Color:**
  - Monster: Yellow/Orange
  - Spell: Green/Teal
  - Trap: Pink/Purple
- **Attribute Symbol:** Upper-right (DARK, LIGHT, FIRE, WATER, WIND, EARTH, DIVINE)
- **Level/Rank Stars:** Upper-left
- **Card Art:** Large central image

### Edge Cases
- **Alternate Art:** Same card number, different art (e.g., tournament prizes)
- **Limited Edition:** Often missing card numbers
- **Promo Cards:** May have `EN###` format instead
- **First Edition:** Red "1st Edition" stamp
- **Misprints:** Wrong set code or missing numbers

## 3. Pokémon TCG

### Card Number Location
- **Position:** Bottom-right corner
- **Format:** `###/###` or `###` (e.g., `25/102`, `SWSH001`)
- **Additional Info:** Set symbol bottom-left, rarity symbol

### Card Number Formats
```
Standard:     25/102
Promo:        SWSH001, XY01
Secret Rare:  103/102 (exceeds set size)
Trainer Gallery: TG01/TG30
```

### Set Code Position
- **Location:** Set symbol bottom-left (visual icon)
- **Format:** Small icon/logo (e.g., Base Set symbol, Sword & Shield logo)

### Variants & Special Versions
- **Rarity Symbols:** (bottom-right, after card number)
  - **●** = Common
  - **◆** = Uncommon
  - **★** = Rare
  - **★H** = Holo Rare
  - **❖** = Ultra Rare (GX, EX, V, VMAX, VSTAR)
- **Reverse Holos:** Foil on everything except art
- **Full Art:** Art covers entire card (textured)
- **Alternate Art:** Different pose/scene, textured
- **Rainbow Rare:** Rainbow holographic (hyper rare)
- **Gold Secret:** Gold foil everything
- **Trainer Gallery:** Special subset (TG##/TG##)

### Visual Distinguishing Features
- **Card Type:**
  - Pokémon: Yellow border (or texture for special)
  - Trainer: Gray border
  - Energy: Varies by type
- **HP:** Upper-right (Pokémon only)
- **Evolution Stage:** Top-left (Basic, Stage 1, Stage 2)
- **Type Symbol:** Upper-right (Fire, Water, Grass, etc.)

### Edge Cases
- **Promo cards:** Black star promo symbol, different numbering
- **World Championship decks:** Different back, gold border
- **Jumbo cards:** Oversized, not tournament legal
- **McDonald's promos:** Different number format
- **Secret rares:** Number exceeds set size

## 4. One Piece Card Game (OPTCG)

### Card Number Location
- **Position:** Bottom-left corner
- **Format:** `XX##-###` (e.g., `ST02-004`, `OP01-001`)
- **Additional Info:** Set code + sequential number

### Card Number Formats
```
Starter Deck: ST##-###
Booster:      OP##-###
Promo:        P-###
Premium:      PRB##-###
```

### Set Code Position
- **Location:** Embedded in card number (bottom-left)
- **Format:** 2-4 letters + 2 digits (e.g., `ST02`, `OP11`)

### Variants & Special Versions
- **Parallel (Foil):** Holographic pattern, same art
- **Alternate Art:** Completely different artwork
- **Full Art:** Extended art beyond frame
- **Jolly Roger Foil:** Special Jolly Roger holographic pattern
- **Manga Variant:** Manga-style artwork
- **Championship Promo:** Tournament exclusive with special stamps

### Visual Distinguishing Features
- **Color:** Card color indicates faction/character
- **Cost:** Upper-left (number)
- **Power:** Upper-right (number)
- **Card Type:** Character, Event, Stage, Leader
- **Rarity Symbol:** Bottom-left (C, UC, R, SR, SEC, L)

### Edge Cases
- **Leader cards:** Different back, different number format
- **DON!! cards:** Special power-up cards
- **Multiple variants per number:** Up to 6+ variants of same card number

## 5. Digimon Card Game

### Card Number Location
- **Position:** Bottom-left corner
- **Format:** `XX#-###` (e.g., `BT1-001`, `ST1-01`)
- **Additional Info:** Set code + rarity indicator

### Card Number Formats
```
Booster:      BT#-###
Starter:      ST#-##
Promo:        P-###
Alternative:  BT#-###_P1 (parallel)
```

### Set Code Position
- **Location:** Embedded in card number (bottom-left)
- **Format:** 2-3 letters + number (e.g., `BT1`, `ST14`)

### Variants & Special Versions
- **Parallel Rare:** Foil treatment, `_P1` suffix
- **Super Rare:** Foil art only
- **Secret Rare:** Full foil with rainbow pattern
- **Alternate Art:** Different art, appended identifier

### Visual Distinguishing Features
- **Color:** Red, Blue, Yellow, Green, Black, Purple, White
- **Level:** Top-left (Rookie, Champion, Ultimate, Mega)
- **DP:** Digimon Power (top-right)
- **Play Cost:** Memory cost at top
- **Digivolution Cost:** Requirements box

### Edge Cases
- **Pre-release promos:** Different back design
- **Tournament pack cards:** Special numbering
- **Double-sided cards:** Rare, two numbers

## 7. Disney Lorcana

### Card Number Location
- **Position:** Bottom-left corner
- **Format:** `###/###` (e.g., `123/204`)
- **Additional Info:** Set symbol, rarity, artist

### Card Number Formats
```
Standard:     123/204
Enchanted:    123/204 (foil + special border)
Promo:        P###
```

### Set Code Position
- **Location:** Set symbol in bottom-left area
- **Format:** Visual symbol (e.g., The First Chapter symbol)

### Variants & Special Versions
- **Foil:** Standard holographic
- **Enchanted:** Special full-art foil (1 per box)
- **Promo:** Store/event exclusives

### Visual Distinguishing Features
- **Ink Color:** Amber, Amethyst, Emerald, Ruby, Sapphire, Steel
- **Cost:** Upper-left circle
- **Ink Value:** Inkwell symbol (if inkable)
- **Card Type:** Character, Action, Item, Location

## Universal Card Number Extraction Strategy

### Multi-Region Scanning Approach
```python
CARD_NUMBER_REGIONS = {
    'bottom_left': (0, 0.85, 0.30, 1.0),      # One Piece, Digimon, Lorcana
    'bottom_center': (0.35, 0.90, 0.65, 1.0), # MTG

}
```

### Universal Pattern Matching
```regex
MTG:         r'\d{1,3}/\d{1,3}[a-z]?'
Yu-Gi-Oh:    r'[A-Z]{3,4}-[A-Z]{2}\d{3}'
Pokémon:     r'(\d{1,3}/\d{1,3}|[A-Z]{2,4}\d{3})'
One Piece:   r'[A-Z]{2,4}\d{2}-\d{3}'
Digimon:     r'[A-Z]{2,3}\d{1,2}-\d{3}'
Gundam:      r'[A-Z]{2}\d{2}-\d{3}'
Lorcana:     r'\d{1,3}/\d{1,3}'
```

## Foil Detection Characteristics

### Visual Signatures by TCG

**MTG:**
- Foil: Consistent rainbow holographic pattern across card
- Etched: Matte textured foil (less shiny)
- Serialized: Gold stamp with number

**Yu-Gi-Oh:**
- Secret Rare: Diagonal rainbow pattern
- Starlight: Starburst sparkle pattern
- Ultimate: Texture/embossing visible in lighting
- Ghost: 3D relief effect

**Pokémon:**
- Reverse Holo: Everything foil except art
- Holo Rare: Sparkle pattern in art only
- Rainbow: Diagonal rainbow stripes
- Texture: Raised texture visible

**One Piece:**
- Parallel: Holographic sheen over entire card
- Jolly Roger: Skull pattern in foil

**Universal Detection:**
- High variance in grayscale
- Specular highlights (bright spots)
- Color iridescence
- Texture/embossing (depth detection)

## Notes for Senior Engineering Implementation

### Challenges to Address:
1. **Ambiguous formats:** Lorcana and MTG both use ###/###
2. **Multiple regions:** Need to scan 3 regions per card
3. **OCR errors:** Foil can obscure text, need robust pattern matching
4. **Language variants:** Yu-Gi-Oh EN/DE/FR codes
5. **Set disambiguation:** Multiple sets can have same number format
6. **Promo cards:** Often non-standard numbering
7. **Alternate arts:** Same number, wildly different appearance

### Architecture Requirements:
- **Cascading region detection:** Try all regions, pick best match
- **Format disambiguation:** Use multiple signals (position + format + visual features)
- **TCG auto-detection:** Infer TCG from card number format
- **Variant clustering:** Group by base card number before scoring
- **Multi-modal scoring:** Visual + geometric + card number match
- **Confidence calibration:** Different thresholds per TCG and variant type
